## 🚀 Cómo ejecutar los proyectos

Si deseas probar alguno de los programas localmente, asegúrate de tener instalado el [SDK de .NET](https://dotnet.microsoft.com/download).

1. Clona el repositorio:
   ```bash
   git clone https://github.com/jacobgtzmtz/ProgramandoEnCSharp.git
   ```
2. Entra en la carpeta del ejercicio:
   ```bash
   cd Seccion2/Variables
   ```
3. Ejecuta el proyecto:
   ```bash
   dotnet run
   ```
